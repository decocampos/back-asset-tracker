# back-asset-tracker
O AssetTracker é uma plataforma da Group Decode Solution para consolidação de ativos globais (Ações, FIIs, Cripto). Oferece dashboards analíticos, cálculo automático de preço médio, suporte multi-moeda e integração de pagamentos via Stripe. Desenvolvido com Node.js e Supabase, foca em segurança, performance e escalabilidade modular.
